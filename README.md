# CC_ComputerVision

## Setup
1. Download Node.js von https://nodejs.org/en/download/
2. In der Konsole den Befehl **npm install -g @angular/cli**

## Starten des Frontends
In der Konsole mit dem Befehl **cd** zum Ordner cv-frontend navigieren und dort **ng serve** ausführen.
 
## Starten des Backendes
Die Datei **CC.AzureVision.REST-API.exe** ausführen. 
Die Pfad zu der Datei lautet wie folgt **CC_ComputerVision\CC.ComputerVision\CC.AzureVision.REST-API\bin\Release\netcoreapp3.1\CC.AzureVision.REST-API.exe**

## Ausführung
Die Webseite öffnet sich auf **Localhost:4200**, hier kann man ein Bild-Datei hochladen (maximal Größe: 4 MB).
Nach der Bearbeitung des Backendes erscheint auf dem Frontend die Beschreibung des Bildes und ein Player für die Audio-Datei.
Als User kann man sich die Audio-Datei runterladen.
