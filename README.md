# CC_ComputerVision
Team : Federhofer, Legat, Rathbauer

## Setup
1. Download Node.js von https://nodejs.org/en/download/
2. In der Konsole die Befehle **npm install -g @angular/cli**,
3. Download des **Backend-Release-1.0.zip** und des **Frontend-Release-1.0.zip**. Diese befinden sich in dem Release Tag.
4. Beide Dateien entpacken. 

## Starten des Frontends
In der Konsole mit dem Befehl **cd** zum Ordner **Frontend-Release-1.0** navigieren und dort **ng serve** ausführen.
 
## Starten des Backendes
Die Datei **CC.AzureVision.REST-API.exe** ausführen. 
Die .exe Datei befidnet sich in den Backend-Release Ordner. In dem **Backup-Release-1.0.zip** File.

## Ausführung
Die Webseite öffnet sich auf **Localhost:4200**, hier kann man ein Bild-Datei hochladen (maximal Größe: 4 MB).
Nach der Bearbeitung des Backendes erscheint auf dem Frontend die Beschreibung des Bildes und ein Player für die Audio-Datei.
Als User kann man sich die Audio-Datei runterladen.
